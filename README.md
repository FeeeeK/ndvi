# NDVI app

Simple app to calculate NDVI from Sentinel-2 imagery using Google Earth Engine API.

## Installation

1. Install poetry

    ```sh
    pip install poetry
    ```

2. Install dependencies

    ```sh
    poetry install
    ```

3. Run the app

    ```sh
    poetry shell
    python -m src
    ```

### Or with docker

 ```sh
 docker compose up
 ```
